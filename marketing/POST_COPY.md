## Blender / artist community version

I released `3DVisual Mesh`, an open-source local image-to-mesh helper for Windows.

The goal is not to pretend it replaces full manual modeling. The goal is to speed up the first step and give artists a workable starting mesh faster.

It is still beta, and I want to be honest that some outputs still need light Blender cleanup. But the project is now live and I am looking for real feedback from artists and technical users.
