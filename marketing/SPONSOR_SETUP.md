# Sponsor Setup for 3DVisual Mesh

## Best choice

Keep `Ko-fi` as the main support link right now.

## Why

- it is already connected to the project
- it is easy for one-time support
- it is the fastest thing for early beta funding

## Second best

Add `GitHub Sponsors` later for recurring monthly support.

## Real downside

I cannot finish the payment and tax setup for you because that part needs your personal details.

## Best sponsor text

If 3DVisual Mesh helps you, you can support development here:

`https://ko-fi.com/3dvisualmesh`

Support is optional and helps fund:

- better mesh quality
- cleanup tools
- Blender workflow improvements
- plugin support
- better NVIDIA coverage
- future creator tools

## Suggested sponsor goals

### Goal 1

Better mesh cleanup and topology polish

### Goal 2

Better preview, texturing, and Blender workflow

### Goal 3

More testing across AMD and NVIDIA hardware

### Goal 4

Community plugin support and better docs

## Suggested page bio

I am building practical open-source creator tools to help solo developers and artists save time. 3DVisual Mesh is the first public step: a local Windows image-to-mesh helper that is still improving and growing with community feedback.

## Best funding model for now

- free beta app
- optional Ko-fi support
- open-source repo
- maybe GitHub Sponsors later

## What I do not recommend yet

- locking the beta behind payment
- hard paywall before the workflow is more mature
- paid ads before launch feedback stabilizes
